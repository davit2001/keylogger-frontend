## Getting Started

## Package installation


Before running this code ensure that you have npm installed

You don't know what is npm. It is okay in that case install Node.js from here
https://nodejs.org/en/download


You have npm in your computer, then run this command
```bash
  npm install
```

## Run project

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
